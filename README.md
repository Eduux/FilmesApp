# Instalação

Entrar na pasta e instalar suas dependências com o comando `npm install`.

## Preview

[https://eduux.github.io/FilmesApp/](https://eduux.github.io/FilmesApp/).

## Para rodar o server

```
npm start
```

## Para rodar os testes

```
npm run test
```

## Para verificar a cobertura de testes

```
npm run coverage
```
